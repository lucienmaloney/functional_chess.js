Standard Representation (Used in the PGN's):
   ---------------------------------
8  |   |   |   |   |   |   |   |   |
   ---------------------------------
7  |   |   |   |   |   |   |   |   |
   ---------------------------------
6  |   |   |   |   |   |   |   |   |
   ---------------------------------
5  |   |   |   |   |   |   |   |   |
   ---------------------------------
4  |   |   |   |   |   |   |   |   |
   ---------------------------------
3  |   |   |   |   |   |   |   |   |
   ---------------------------------
2  |   |   |   |   |   |   |   |   |
   ---------------------------------
1  |   |   |   |   |   |   |   |   |
   ---------------------------------
     A   B   C   D   E   F   G   H

Ex: A1, D5, H3, B3

   ---------------------------------

   ---------------------------------
1  |   |   |   |   |   |   |   |   |
   ---------------------------------
     1   2   3   4   5   6   7   8

Ex: 11, 45, 83, 23

Add 10 to move up one square, subtract 10 to move down
Add  1 to move right one sqr, subtract  1 to move left

Every square number will be two digit, and should not contain a 0 or a 9
